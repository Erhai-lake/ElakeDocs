GitHub/Gitee 仓库卡片

约 248 字小于 1 分钟

2025-12-25

Repo卡片组件用于显示GitHub/Gitee仓库信息.

<RepoCard repo="pengzhanbo/vuepress-theme-plume" />

repoRequiredstring

仓库地址, 格式为owner/repo.

providerOptionalgithub | gitee

github

仓库所属平台, 当前仅支持github/gitee.

fullnameOptionalboolean

是否显示完整的仓库名称.

完整的仓库名称为owner/repo.

  • 如果owner为个人, 则默认不显示完整的仓库名称, 仅显示repo.
  • 如果owner为组织, 则默认显示完整的仓库名称.

示例

单卡片

多卡片

如果希望以紧凑的方式并排展示多个卡片, 可以使用CardGrid组件.

<CardGrid>
	<RepoCard repo="vuepress/marketplace" />
	<RepoCard repo="pengzhanbo/vuepress-theme-plume" />
</CardGrid>
<CardGrid>
  <RepoCard repo="pengzb/vuepress-theme-plume" provider="gitee" />
  <RepoCard repo="Erhai-lake/ElakeDocs" provider="gitee" />
</CardGrid>
