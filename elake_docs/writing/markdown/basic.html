表格

在 Markdown 中, 表格的对齐方式默认是左对齐. 要改变对齐方式, 请在表头行中使用冒号(:)来指定对齐方向.

  • 冒号在表头行的左侧, 表示左对齐.
  • 冒号在表头行的右侧, 表示右对齐.
  • 冒号在表头行的两侧, 表示居中对齐.
