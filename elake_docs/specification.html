文档编写规范.md

文档必须有frontmatter

标题会渲染到文本顶部和左边侧边栏的标题

---
title: 这里写标题, 例如 文档编写规范
createTime: 这里写时间, 例如 2025/12/22 12:48:13
---

版权

这是一个很重要的规范, 故写在frontmatter规范下面

在frontmatter中添加copyright字段, 默认是CC-BY-4.0, 可以不用写.

文档自带的版权

  • CC0 1.0 通用 (CC0)
  • 署名 4.0 国际 (CC-BY-4.0)
  • 署名-相同方式共享 4.0 国际 (CC-BY-SA-4.0)
  • 署名-非商业性 4.0 国际 (CC-BY-NC-4.0)
  • 署名-禁止演绎 4.0 国际 (CC-BY-ND-4.0)
  • 署名-非商业性-相同方式共享 4.0 国际 (CC-BY-NC-SA-4.0)
  • 署名-非商业性-禁止演绎 4.0 国际 (CC-BY-NC-ND-4.0)

代码块内的内容需在编辑器中格式化

这样看起来好看一点, 并且和其他文档格式保持一致

正例
```javascript
let i = 1;
while (i <= 3) {
	console.log(`这是我循环的第${i}遍`)
	i++
}
```
反例
```javascript
let i=1;
while(i<=3){
console.log(`这是我循环的第${i}遍`)
 i++
}
```

代码块缩进大小为制表符4

正例
```javascript
let i = 1;
while (i <= 3) {
	console.log(`这是我循环的第${i}遍`)
	i++
}
```
反例
```javascript
let i = 1
while (i <= 3) {
  console.log(`这是我循环的第${i}遍`)
  i++
}
```
