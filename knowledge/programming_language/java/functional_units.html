功能单元

约 113 字小于 1 分钟

2025-12-27

注意

以下代码我知道你还没学.

现在是整体学习, 不要求你能看懂以下代码!

public static void printHelloWorld() {
    System.out.println("Hello World");
}

其中printHelloWorld就是这个功能的名字, 也叫方法名, 如果以后要使用这个功能, 就根据功能的名字找到这个功能运行.

例子:

public static void main(String[] args) {
    printHelloWorld();
}

public static void printHelloWorld() {
    System.out.println("Hello World");
}
