列表标签

约 277 字小于 1 分钟

2024-10-21

作用: 布局内容排列整齐的区域.

列表分为 3 种无序列表, 有序列表, 定义列表.

无序列表

作用: 布局排列整齐的不需要规定顺序的区域.

标签名: ul, li双标签.

<ul>
    <li>表项一</li>
    <li>表项二</li>
    <li>表项三</li>
</ul>

注意

  • ul标签里面只能嵌套li标签.
  • li标签里面可以嵌套如何内容.

有序列表

作用: 布局排列整齐的需要规定顺序的区域.

标签名: ol, li双标签.

<ol>
    <li>表项一</li>
    <li>表项二</li>
    <li>表项三</li>
</ol>

注意

  • ol标签里面只能嵌套li标签.
  • li标签里面可以嵌套如何内容.

定义列表

标签名: dl, dt, dd双标签.

<dl>
    <dt>列表标题</dt>
    <dd>表项一</dd>
    <dd>表项二</dd>
    <dd>表项三</dd>
</dl>

注意

  • dl标签里面只能嵌套dt标签和dd标签.
  • dt标签和dd标签里面可以嵌套如何内容.
