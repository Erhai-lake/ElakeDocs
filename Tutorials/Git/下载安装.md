# 下载安装

[Git的官网](https://git-scm.com/)

默认配置即可

安装完成后,鼠标右键菜单会出现两项:

1. Git GUI Here
2. Git Bash Here

一个是Git提供的图形界面工具

另一个是Git提供的命令行工具,内置了小的Linux指令(可以用CMD)
