# 在VSCode使用Git

## 在VSCode配置Git

安装好 VSCode 和 Git 后,VSCode会自动识别Git的位置,如果没有自动识别到,则需要手动配置

![6-1](assets/6-1.png)

点击后会打开一个`settings.json`的文件,并且会自动写入配置

![6-2](assets/6-2.png)

如果没有请自行输入:

```json
"git.path": "Git路径,例如:C:/Program Files/Git/cmd/git.exe"
```
