# 数组

数组(Array),是一种将**一组数据存储在单个变量下**的存储方式(优雅~)

在上一篇已经讲述了如何声明变量了,这篇就不重复讲一遍了

## 声明语法

```js
let 变量名 = []

// 例子:
let Name = ['小米', '小明', '张三', '李四', '小天']
```

## 读取数组

```js
// 打印整个数组
console.log(Name)
// 输出 ['小米', '小明', '张三', '李四', '小天']

// 打印数组里的其中一项
console.log(name[0])
// 输出 小米

console.log(name[4])
// 输出 小天
```

在数组中,数据的编号,也就是什么的0和4,被称为**索引**,或者**下标**

数组第一项的索引是0,第二项的索引是1,以此类推

## 一些术语

元素:数组中保存的每个数据都叫数组元素

索引,下标:数组中数据的编号

长度:数组中时数据的个数,可以通过数组的length属性获取`console.log(name.length)`
