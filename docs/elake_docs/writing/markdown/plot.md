有时候, 你不想直接把文本内容毫无保留的展示出来, 想要保留一些隐秘性, 可能是为了引起读者的好奇心, 也可能纯粹是故意增加点阅读障碍, 让文章变得更加有趣.

你知道么, !!鲁迅!! 是一个中国著名的思想家, 他的思想对中国社会产生了深远的影响.

输入

```markdown
遮罩层效果 + 鼠标悬停: !!需要隐秘的内容!!{.mask .hover}

遮罩层效果 + 点击: !!需要隐秘的内容!!{.mask .click}

文本模糊效果 + 鼠标悬停: !!需要隐秘的内容!!{.blur .hover}

文本模糊效果 + 点击: !!需要隐秘的内容!!{.blur .click}
```

::: demo-wrapper

遮罩层效果 + 点击: !!需要隐秘的内容!!{.mask .click}

文本模糊效果 + 鼠标悬停: !!需要隐秘的内容!!{.blur .hover}

文本模糊效果 + 点击: !!需要隐秘的内容!!{.blur .click}
:::

看懂了么

* {.mask} 是遮罩层效果
* {.blur} 是文本模糊效果
* {.hover} 是鼠标悬停效果
* {.click} 是点击效果
