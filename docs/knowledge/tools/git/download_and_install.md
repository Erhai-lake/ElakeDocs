[Git 的官网](https://git-scm.com)

默认配置即可.

安装完成后, 鼠标右键菜单会出现两项:

1. Git GUI Here.
2. Git Bash Here.

一个是**Git**提供的图形界面工具.

另一个是**Git**提供的命令行工具, 内置了小的**Linux**指令(可以用**CMD**).

## 配置 Git

设置用户信息.

```shell
git config --global user.name "用户名"
```

```shell
git config --global user.email "邮箱"
```

查看用户信息.

```shell
git config --global user.name
```

```shell
git config --global user.email
```
