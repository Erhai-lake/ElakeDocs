createTime: 2024/10/02 14:53:07
---

作用: 鼠标==悬停==在元素上时, ==指针显示的样式==.

属性名: `cursor`.

属性值:

|    属性值    |       效果        |
|:---------:|:---------------:|
| `default` |    默认值, 箭头.     |
| `pointer` |  小手, 提示用户可以点击.  |
|  `text`   | 工字, 提示用户可以输入文字. |
|  `move`   |  十字, 提示用户可以移动.  |
