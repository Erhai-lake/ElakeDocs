title: 垂直对齐方式
createTime: 2024/10/02 14:53:07
---

属性名: `vertical-align`.

属性值:

`baseline`: 基线对齐(默认).

`top`: 顶部对齐.

`middle`: 居中对齐.

`bottom`: 底部对齐.

例子:

::::demo

:::code-tabs
@tab:active HTML
```html
<div>
    <img src="assets/304ba504f27b147a71e00dd072a6be3dbb43138e.gif" width="300" alt="img">
    Test
</div>
```

@tab CSS
```css
div {
    border: 1px solid black;
}

img {
    vertical-align: middle;
}
```
