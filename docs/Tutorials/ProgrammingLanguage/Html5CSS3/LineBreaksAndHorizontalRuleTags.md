# 换行与水平线标签

## 换行标签

标签名:`br`单标签

```html showLineNumbers live
<html>
    <body>
        <p>段落1<br />段落2<br />段落3</p>
    </body>
</html>
```

## 水平线标签

标签名:`hr`单标签

```html showLineNumbers live
<html>
    <body>
        <p>段落1</p>
        <hr />
        <p>段落2</p>
        <hr />
        <p>段落3</p>
    </body>
</html>
```
