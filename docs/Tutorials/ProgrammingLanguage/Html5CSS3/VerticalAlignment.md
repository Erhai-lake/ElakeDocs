# 垂直对齐方式

属性名:`vertical-align`

属性值:

`baseline`:基线对齐(默认)

`top`:顶部对齐

`middle`:居中对齐

`bottom`:底部对齐

例子:

```html showLineNumbers
<div>
    <img src="image.png" width="300">
    Test
</div>
```

```css showLineNumbers
div {
    border: 1px solid black;
}

img {
    vertical-align: middle;
}
```
