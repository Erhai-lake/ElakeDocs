# vw单位

vw单位,是**相对视口宽度**的计算结果,可以直接实现移动端适配效果

1vw=1/100视口宽度

例子:

```html showLineNumbers
<div></div>
```

```css showLineNumbers
div {
    width: 50vw;
    height: 50vw;
    background-color: red;
}
```
