# Mods信息列表

|        属性        |                               值                                |
| :----------------: | :-------------------------------------------------------------: |
|       属性名       |                         `ModsInfoList`                          |
|        `ID`        |                          模组ID 字符串                          |
|       `Name`       |                         模组名称 字符串                         |
|    `WebsiteUrl`    |                         平台地址 字符串                         |
|     `Summary`      |                           总结 字符串                           |
|  `DownloadCount`   |                          下载量 长整型                          |
|     `Authors`      | 作者信息列表 列表([`AuthorInfoList`](../Return/AuthorInfoList)) |
| `LogoThumbnailUrl` |                          缩略图 字符串                          |
|     `LogoUrl`      |                           原图 字符串                           |
|   `DateCreated`    |                         创建时间 字符串                         |
|   `DateModified`   |                         修改时间 字符串                         |
|   `DateReleased`   |                     发布时间(MR没有) 字符串                     |
