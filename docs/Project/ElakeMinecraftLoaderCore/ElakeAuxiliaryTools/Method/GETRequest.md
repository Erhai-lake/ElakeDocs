# 网络请求

|    属性    |            值            |
| :--------: | :----------------------: |
|   方法名   |      `GETRequest()`      |
|    描述    | 请求URL链接,返回请求结果 |
| 参数 `URL` |       链接 字符串        |
|    返回    |    RAM信息数组 字符串    |

## 例子

<Tabs>
    <TabItem value="Code" label="代码">
        ```C# showLineNumbers
        private void AuxiliaryTools()
        {
            Console.WriteLine(await ElakeAuxiliaryTools.GETRequest("https://api.elake.top/"));
        }
        ```
