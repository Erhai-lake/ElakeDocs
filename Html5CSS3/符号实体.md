# 符号实体

作用:在网页中 **显示预留字符**

列出一些常用的:

|    字符     |     预留     |
| :---------: | :----------: |
|  `&nbsp;`   | &nbsp;(空格) |
|   `&lt;`    |     &lt;     |
|   `&gt;`    |     &gt;     |
|   `&le;`    |     &le;     |
|   `&ge;`    |     &ge;     |
|  `&#9745;`  |   &#9745;    |
| `&#10003;`  |   &#10003;   |
| `&#10005;`  |   &#10005;   |
| `&#10007;`  |   &#10007;   |
|  `&copy;`   |    &copy;    |
| `&#128512;` |  &#128512;   |
